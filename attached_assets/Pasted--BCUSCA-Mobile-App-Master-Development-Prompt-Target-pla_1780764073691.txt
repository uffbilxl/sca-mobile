# BCUSCA Mobile App — Master Development Prompt

> **Target platforms:** iOS (App Store) & Android (Google Play)
> **Source website:** [bcusca.org](https://bcusca.org)
> **Framework recommendation:** React Native (Expo) for cross-platform parity

---

## 1. Project Overview

Build a **pixel-faithful mobile application** of the BCU Student Computing Association website (bcusca.org) for release on the **Apple App Store** and **Google Play Store**. Every screen, section, piece of content, navigation item, and interaction must mirror the live website exactly — no content additions, no content removals, no layout deviations — with one deliberate exception: a **professional animated splash screen** shown on cold launch before the home screen appears.

This app is built by BCU students, for BCU students. It must feel native, fast, and polished across both platforms.

## 2. Typography — Match the Website Exactly

The website uses **Geist Sans** by Vercel, loaded via `next/font/google`. This must be the font used throughout the entire app, including the splash screen.

- **Primary font:** Geist Sans (variable font, weights 100–900)
- **Monospace font:** Geist Mono (for any code or terminal-style text)
- **Install in React Native:** Use `expo-font` with self-hosted Geist font files from https://github.com/vercel/geist-font/releases
- **Fallback stack:** `'Geist', system-ui, -apple-system, sans-serif`

### Font weight usage (match site exactly)
| Context | Weight |
|---|---|
| Hero heading "Your BCU Computing" | 700 (Bold) |
| Hero sub-heading "Community & Career Hub" | 400–500 |
| Section headings | 600 (SemiBold) |
| Sub-labels ("HANDPICKED", "01") | 400, uppercase, tracked |
| Body copy | 400 (Regular) |
| Navigation items | 500 (Medium) |
| Badges (Featured, Hybrid, etc.) | 500, small |
| CTA button text | 600 |

### Geist font setup in Expo
```javascript
// app/_layout.tsx
import { useFonts } from 'expo-font';

const [fontsLoaded] = useFonts({
  'Geist-Light':    require('../assets/fonts/Geist-Light.ttf'),
  'Geist-Regular':  require('../assets/fonts/Geist-Regular.ttf'),
  'Geist-Medium':   require('../assets/fonts/Geist-Medium.ttf'),
  'Geist-SemiBold': require('../assets/fonts/Geist-SemiBold.ttf'),
  'Geist-Bold':     require('../assets/fonts/Geist-Bold.ttf'),
});
```

## 4. Splash Screen Specification

The splash screen is the **only intentional difference** from the website. It uses the **exact same font (Geist Sans) and colour scheme (dark navy + teal/purple glows)** as the site.

### Purpose
The splash screen exists solely to greet the user on cold launch. It contains **no buttons, no navigation, no interactive elements of any kind**. It displays, then automatically transitions to the Home screen. The user does nothing — it just plays and moves on.

### Design
- **Background:** Deep navy `#0d1117` — same as site base background
- **Ambient glows:** Teal glow (bottom-left) + purple glow (bottom-right) — matching the hero section exactly
- **Font:** Geist Sans throughout — same as the entire website
- **Logo:** SCA logo (`https://bcusca.org/sca-logo.png`) centred, scales in from 85%→100% with fade over 600ms
- **BCU pill:** Small pill label "BIRMINGHAM CITY UNIVERSITY · STUDENT COMPUTING ASSOCIATION" — matching the exact pill on the website hero (border `#30363d`, green dot `#3fb68b`, text `#8b949e`, Geist 400)
- **Main heading:** "Student Computing Association" — Geist Bold 700, white `#ffffff`
- **Tagline:** `"From your first lecture to your first offer."` — Geist Light 300, `rgba(255,255,255,0.45)`, italic
- **Ticker strip:** Company names scrolling at bottom at ~20% opacity — ambient decoration only, text `rgba(139,148,158,0.35)`
- **No buttons. No CTAs. No tappable elements.** The screen is purely visual.
- **No gold, no purple text, no other accent** — all colour comes from the background glows only

### Animation sequence
| Time | Event |
|---|---|
| 0ms | Dark navy screen + ambient glows present immediately |
| 100ms | Logo scales + fades in (600ms duration) |
| 700ms | BCU pill fades up |
| 950ms | "Student Computing Association" heading fades up |
| 1200ms | Tagline fades up |
| 0ms (bg) | Ticker scrolling begins immediately |
| 2800ms | All elements fade out to `#0d1117` |
| 3100ms | Crossfade into Home screen |

### Platform compliance
- **iOS:** Safe Area Insets respected. `LaunchScreen.storyboard` background = `#0d1117`. Centred SCA logo as launch image.
- **Android:** `windowSplashScreenBackground` = `#0d1117`. Custom animated splash activity overlaid immediately after system splash. Glows rendered via Canvas or pre-baked PNG.
- **Both:** Splash does NOT replay on app resume — cold launch only.
